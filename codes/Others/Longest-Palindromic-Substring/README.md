## [最长回文子串](https://leetcode-cn.com/problems/longest-palindromic-substring/)

给你一个字符串<code>s</code>，找到<code>s</code>中最长的回文子串。

```
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```

```
输入：s = "cbbd"
输出："bb"
```

```
输入：s = "a"
输出："a"
```

```
输入：s = "ac"
输出："a"
```
