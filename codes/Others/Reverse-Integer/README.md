给你一个<code>32</code>位的有符号整数<code>x</code> ，返回将<code>x</code>中的数字部分反转后的结果。如果反转后整数超过<code>32</code>位的有符号整数的范围<code>[−2<sup>31</sup>, 2<sup>31</sup>−1]</code>，就返回 0。

**假设环境不允许存储<code>64</code>位整数（有符号或无符号）。**

```
输入：x = 123
输出：321
```

```
输入：x = -123
输出：-321
```

```
输入：x = 120
输出：21
```

```
输入：x = 0
输出：0
```
