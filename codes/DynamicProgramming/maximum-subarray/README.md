## [最大子序和](https://leetcode-cn.com/problems/maximum-subarray/)

给定一个整数数组<code>nums</code>，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

```
输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
输出：6
解释：连续子数组 [4,-1,2,1] 的和最大，为 6 。
```

```
输入：nums = [1]
输出：1
```

```
输入：nums = [-1]
输出：-1
```
